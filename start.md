# Role & Objective
Act as a Senior UI/UX Designer and Expert Frontend Developer. Your goal is to architect and code a high-end, pixel-perfect **Landing Page (First Page)** using a modern "Antigravity" design aesthetic (clean layouts, fluid animations, and depth).

# Tech Stack & Constraints
- **Core:** Use ONLY vanilla HTML5, CSS3, and modern JavaScript (ES6+). 
- **No Frameworks:** Strictly no React, Vue, Tailwind, or Bootstrap. 
- **Semantic HTML:** Use proper tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
- **Modern Styling:** - Use CSS Variables (`:root`) for the entire design system (colors, scales, transitions).
  - Layouts must use CSS Grid and Flexbox.
  - Implement fluid typography using `clamp()`.
  - Use `backdrop-filter` for glassmorphism and modern layering.
- **Responsiveness:** Ensure a seamless mobile-first responsive transition.

# Design & Assets Instructions
1. **Visual Identity:** - **Colors:** Define a professional palette (Background, Surface, Primary, Secondary, and High-Contrast Text).
   - **Typography:** Use Google Fonts (e.g., 'Inter', 'Plus Jakarta Sans', or 'Syne'). Establish a clear typographic hierarchy.
   - **Spacing & Radius:** Use a consistent 8px/4px grid system for margins, padding, and `border-radius`.
2. **Components to Build:**
   - **Navigation:** Sticky glass-effect header with active state links.
   - **Hero Section:** High-impact headline, subtext, and primary/secondary Call-to-Action buttons.
   - **Features/Services:** A 3 or 4-column grid with hover-animated cards.
   - **Social Proof:** A section for testimonials or client logos.
3. **Icons:** Use **Iconify (Solar Linear)** via CDN for all UI elements.
4. **Interactivity:** Implement smooth scroll, subtle fade-in animations on scroll (via Intersection Observer), and polished hover states for all interactive elements.

# Output Requirements
- Provide the complete code in a structured format (HTML, CSS, JS).
- Add brief, professional comments explaining the layout logic.
- Ensure the final result looks like a premium, production-ready landing page.
